# catnip

**catnip is a tiny self-care companion — check in on yourself, and your cat grows.** 오늘 하루를 돌보러 오면, 쓰다듬으면 찰랑거리는 고양이가 곁에 있습니다 — 이미지 없이 전부 코드로 그린 인터랙티브 플러시 퍼.

정적 페이지 한 장(`index.html`)이 전부입니다. 빌드 과정 없음.

- **오늘의 돌봄 체크인** — 하루 컨디션을 기록하면 고양이가 반응하고 애정도가 쌓임
- **조용한 방** — 고양이가 곁에서 자는 동안 집중하는 포커스 타이머, 완주하면 젤리 보상
- **털** — 마리당 ~25,000가닥을 WebGL2 인스턴싱으로 `drawArraysInstanced` 한 번에 렌더
- **물리** — 64×36 감쇠 스프링 필드 한 장. 커서가 지나가면 결이 갈라졌다가 잔물결로 복원
- **꼬리·수염** — verlet 로프 + 뿌리→끝 강성 기울기
- **품종** — 치즈·고등어·삼색이·턱시도, 코트는 가닥에 색을 주는 함수 하나 차이
- **쉬어가는 놀이 8종** — 레이저 포인터, 낚싯대, 꾹꾹이, 상자, 그루밍 등 잠깐 쉬어가는 미니게임

## 조작

- 마우스로 몸을 쓰다듬기 / 꼬리·수염 건드리기
- `←` `→` 또는 스와이프로 4마리 넘기기

## 로컬 실행

정적 파일이라 아무 서버나 됩니다.

```
python3 -m http.server
```

WebGL2 필요.
